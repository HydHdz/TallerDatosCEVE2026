## Datos utilizados:

- CMS Collaboration, McCauley, Thomas (s.f.). Datasets derived from the Run2011A SingleElectron, SingleMu, DoubleElectron, and DoubleMu primary datasets. Data recorded in 2011 and published in 2017. https://opendata.cern.ch/record/545
- CMS Collaboration (2024). DoubleMuon primary dataset in NANOAOD format from RunG of 2016 (/DoubleMuon/Run2016G-UL2016_MiniAODv2_NanoAODv9-v2/NANOAOD). CERN Open Data Portal. DOI:10.7483/OPENDATA.CMS.ZQS3.LGLP
